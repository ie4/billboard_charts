# BillboardCharts

get billboard chart

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'billboard_charts'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install billboard_charts

## Usage

	hot100 = BillboardCharts.new.get_list
